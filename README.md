# Magic_Mirror
Magic Mirror - программное обеспечение для студенческого проекта "Волшебное зеркало".
Реализованные  функции:
- Получение и вывод информации о погоде в указанном городе
- Получение и вывод информации о дате и времени в соответствии с указанным часовым поясом
- Вывод указанных заметок
- Управление через бота в Telegram
